# Movie App (React + Node/MongoDB)

A simple movie management application built with **React** (frontend) and **Node.js + Express + MongoDB** (backend).  
You can add movies, list them, and submit reviews with ratings and comments.

##  Features
- Add new movies with title, year, and genre  
- Retrieve details of a movie 
- Find movie details by title  
- Add reviews with rating (0–5) and comments  
- Find top rated movies

## Folder structure:
This project contains two main folders:
- **`API/`** → Backend (Node.js, Express, MongoDB)  
- **`movies-ui/`** → Frontend (React)  

## How to run:
Navigate to the desired folder (`API` or `movies-ui`)  
Follow the instructions in that folder’s **README.md**
